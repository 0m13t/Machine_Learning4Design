# Machine_Learning4Design
This project was completed as part of a university course and was carried out in a group.

The goal of the project was to analyse anonymous student feedback about the COVID-era learning experience at TU Delft. We used text processing and machine learning techniques to identify recurring themes, emotions and student needs.

The analysis included text preprocessing, word frequency analysis, sentiment analysis and topic modelling. These methods helped reveal key issues such as lack of social contact, reduced motivation, online learning challenges and the need for better student support.

Here I learned the data processing and preparation on text based data.
